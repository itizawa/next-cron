# next-cron



## Description

Fulfill a cron job using Express Custom Server with Next.js.

## Features

- timeSignalService
  - Send slack notification once an hour.
- youtubeService
  - Add new videos of channels registered in a specific playlist.
- githubGrassService
  - At the end of the day you will be notified of the number of contributions for today.
  - Notifies me of the report once a week.
- qiitaTrendService
  - Notify qiita articles on a particular topic once every **15 minutes**.
